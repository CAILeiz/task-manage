## MODIFIED Requirements

### REQ-001: 侧边栏布局

**优先级**: P1

在页面左侧添加固定宽度侧边栏。**在移动端改为抽屉式导航。**

**验收标准**:
- [ ] 桌面端侧边栏宽度 240px（展开）/ 64px（折叠）
- [ ] 固定在左侧，不随页面滚动
- [ ] 浅色背景区分主内容区
- [ ] 包含搜索框、快捷视图、标签区域
- [ ] **移动端（< 768px）：隐藏固定侧边栏，改为抽屉式导航**
- [ ] **移动端：抽屉从左侧滑入，带有遮罩层**

**UI规格**:
```
桌面端：
┌──────────────┬────────────────────────────┐
│ 🔍 搜索...   │                            │
│              │                            │
│ ○ 所有任务   │        主内容区            │
│ ○ 今日 (3)   │                            │
└──────────────┴────────────────────────────┘

移动端：
┌────────────────────┐
│ ☰  任务管理   👤  │  ← Header
├────────────────────┤
│                    │
│    主内容区        │  ← 全宽
│                    │
│                  + │  ← FAB
└────────────────────┘

点击汉堡菜单后：
┌──────────────┬─────┐
│ 🔍 搜索...   │     │
│              │     │
│ ○ 所有任务   │ /// │  ← 遮罩层
│ ○ 今日 (3)   │     │
└──────────────┴─────┘
```

### REQ-002: 快捷视图入口

**优先级**: P1

侧边栏提供快捷视图切换入口。**移动端通过抽屉访问。**

**验收标准**:
- [ ] 所有任务：显示全部任务计数
- [ ] 今日任务：显示今日到期计数
- [ ] 过期任务：显示过期任务计数（红色徽章）
- [ ] 即将到期：显示7天内到期任务
- [ ] 无期限：显示无截止日期任务
- [ ] 当前激活项高亮显示
- [ ] **移动端：点击抽屉中的导航项后自动关闭抽屉**

#### Scenario: 移动端导航切换
- **WHEN** 用户在移动端抽屉中点击导航项
- **THEN** 视图切换到对应内容，抽屉自动关闭

### REQ-003: 搜索功能

**优先级**: P2

侧边栏顶部提供任务搜索。**移动端通过抽屉访问。**

**验收标准**:
- [ ] 输入框带搜索图标
- [ ] 实时筛选任务列表
- [ ] 搜索标题和描述内容
- [ ] 清空搜索框恢复全部显示
- [ ] **移动端：搜索框在抽屉内显示**

### REQ-004: 标签筛选

**优先级**: P2

侧边栏底部显示标签列表。**移动端通过抽屉访问。**

**验收标准**:
- [ ] 显示用户创建的标签
- [ ] 点击标签筛选对应任务
- [ ] 标签显示颜色标识
- [ ] 支持多标签筛选
- [ ] **移动端：标签列表在抽屉内显示**

## Technical Constraints

### TC-001: 组件要求
- 使用 Element Plus `el-aside`, `el-menu`, `el-input`, `el-badge`, `el-drawer` 组件
- 响应式处理：小屏幕（< 768px）使用抽屉模式
- 使用 `useBreakpoint` composable 检测设备类型

### TC-002: 路由集成
- 视图切换通过 URL query 参数实现
- 支持 `/tasks?view=today` 直接访问
- 移动端切换视图后关闭抽屉
