# Sidebar Navigation Specification

## Capability

侧边栏导航，提供快捷视图入口和标签筛选。

## Requirements

### REQ-001: 侧边栏布局

**优先级**: P1

在页面左侧添加固定宽度侧边栏。

**验收标准**:
- [ ] 侧边栏宽度 200px
- [ ] 固定在左侧，不随页面滚动
- [ ] 浅色背景区分主内容区
- [ ] 包含搜索框、快捷视图、标签区域

**UI规格**:
```
┌──────────────┬────────────────────────────┐
│ 🔍 搜索...   │                            │
│              │                            │
│ ○ 所有任务   │        主内容区            │
│ ○ 今日 (3)   │                            │
│ ○ 过期 (1)   │                            │
│ ○ 即将到期   │                            │
│ ○ 无期限     │                            │
│              │                            │
│ ──────────   │                            │
│ 标签         │                            │
│  🏷 工作     │                            │
│  🏷 个人     │                            │
└──────────────┴────────────────────────────┘
```

### REQ-002: 快捷视图入口

**优先级**: P1

侧边栏提供快捷视图切换入口。

**验收标准**:
- [ ] 所有任务：显示全部任务计数
- [ ] 今日任务：显示今日到期计数
- [ ] 过期任务：显示过期任务计数（红色徽章）
- [ ] 即将到期：显示7天内到期任务
- [ ] 无期限：显示无截止日期任务
- [ ] 当前激活项高亮显示

### REQ-003: 搜索功能

**优先级**: P2

侧边栏顶部提供任务搜索。

**验收标准**:
- [ ] 输入框带搜索图标
- [ ] 实时筛选任务列表
- [ ] 搜索标题和描述内容
- [ ] 清空搜索框恢复全部显示

### REQ-004: 标签筛选（P2）

**优先级**: P2

侧边栏底部显示标签列表。

**验收标准**:
- [ ] 显示用户创建的标签
- [ ] 点击标签筛选对应任务
- [ ] 标签显示颜色标识
- [ ] 支持多标签筛选

## Technical Constraints

### TC-001: 组件要求
- 使用 Element Plus `el-aside`, `el-menu`, `el-input`, `el-badge` 组件
- 响应式处理：小屏幕可折叠

### TC-002: 路由集成
- 视图切换通过 URL query 参数实现
- 支持 `/tasks?view=today` 直接访问

## Out of Scope

- 侧边栏可折叠功能
- 自定义侧边栏排序
- 侧边栏宽度调整
